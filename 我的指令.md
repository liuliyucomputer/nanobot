# nanobot 多轮对话使用指南

## 1. 交互模式（推荐）
```bash
# 进入交互模式，可以连续对话
./venv/bin/nanobot agent
```

## 2. 指定会话ID进行多轮对话
```bash
# 使用指定会话ID进行对话，会记住上下文
./venv/bin/nanobot agent -s "my_session" -m "第一句话"

# 继续在同一会话中对话
./venv/bin/nanobot agent -s "my_session" -m "你还记得我吗？"
```

## 3. 默认会话
```bash
# 使用默认会话ID (cli:default)
./venv/bin/nanobot agent -m "消息1"
./venv/bin/nanobot agent -m "消息2"  # 会记住消息1的上下文
```

## 4. 会话管理
- 每个会话ID都有独立的记忆和上下文
- 会话信息保存在 `/root/.nanobot/workspace/` 目录
- 可以同时维护多个不同主题的会话

## 5. 实用示例
```bash
# 学习会话
./venv/bin/nanobot agent -s "learning" -m "我想学习Python"

# 工作会话
./venv/bin/nanobot agent -s "work" -m "帮我分析这个项目"

# 继续学习会话
./venv/bin/nanobot agent -s "learning" -m "刚才我们学到哪里了？"
```

# nanobot 技能使用指南

## 1. 天气查询
```bash
# 查询天气
./venv/bin/nanobot agent -m "今天北京的天气怎么样？"
./venv/bin/nanobot agent -m "明天上海会下雨吗？"
```

## 2. GitHub 操作
```bash
# GitHub 相关操作（需要先安装 gh CLI）
./venv/bin/nanobot agent -m "帮我查看我的 GitHub 仓库"
./venv/bin/nanobot agent -m "创建一个新的 GitHub issue"
./venv/bin/nanobot agent -m "列出最近的 GitHub 提交"
```

## 3. 内容总结
```bash
# 总结网页内容
./venv/bin/nanobot agent -m "总结这个网页的内容：https://example.com"

# 总结 YouTube 视频
./venv/bin/nanobot agent -m "总结这个 YouTube 视频：https://youtube.com/watch?v=xxx"

# 总结文件
./venv/bin/nanobot agent -m "总结文件 /path/to/file.txt 的内容"
```

## 4. tmux 会话管理
```bash
# tmux 操作
./venv/bin/nanobot agent -m "列出当前的 tmux 会话"
./venv/bin/nanobot agent -m "创建一个新的 tmux 窗口"
./venv/bin/nanobot agent -m "在 tmux 中运行命令"
```

## 5. 创建新技能
```bash
# 创建自定义技能
./venv/bin/nanobot agent -m "帮我创建一个新技能，用于管理待办事项"
./venv/bin/nanobot agent -m "创建一个数据分析技能"
```

## 6. 网络搜索（需要配置 API）
```bash
# 配置搜索 API 后使用
./venv/bin/nanobot agent -m "搜索最新的 AI 技术趋势"
./venv/bin/nanobot agent -m "查找 Python 最佳实践"
```

## 7. 文件操作
```bash
# 文件读写
./venv/bin/nanobot agent -m "读取文件 config.json 的内容"
./venv/bin/nanobot agent -m "创建一个 Python 脚本文件"
./venv/bin/nanobot agent -m "分析这个日志文件"
```

## 8. 系统命令
```bash
# 执行系统命令
./venv/bin/nanobot agent -m "查看当前目录的文件"
./venv/bin/nanobot agent -m "检查系统状态"
./venv/bin/nanobot agent -m "安装 Python 包"
```

## 9. 定时任务管理
```bash
# 添加定时任务
./venv/bin/nanobot cron add --name "daily_report" --message "生成每日报告" --cron "0 9 * * *"

# 列出定时任务
./venv/bin/nanobot cron list

# 删除定时任务
./venv/bin/nanobot cron remove <job_id>

# 手动运行任务
./venv/bin/nanobot cron run <job_id>
```

## 10. 状态检查
```bash
# 检查 nanobot 状态
./venv/bin/nanobot status

# 检查渠道状态
./venv/bin/nanobot channels status
```

## 11. 启动网关（用于移动端）
```bash
# 启动网关服务
./venv/bin/nanobot gateway

# WhatsApp 登录
./venv/bin/nanobot channels login
```

## 技巧提示
- 使用交互模式进行连续对话更方便
- 为不同主题使用不同的会话ID
- 技能会根据对话内容自动调用
- 可以要求 nanobot 解释如何使用某个特定技能
